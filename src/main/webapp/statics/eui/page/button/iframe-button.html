按钮

为您提供了下列类型的“按钮”：

  • 重要按钮
  • 一级按钮
  • 二级按钮
  • 冻结按钮
  • 图标按钮
  • 图片按钮

页面引入资源

   <!-- 基本引入  -->
                                <link href="../../plugins/bootstrap-3.3.7-dist/css/bootstrap.min.css" rel="stylesheet">
                                <script src="../../js/jquery-2.1.1.js"></script>
                                <script src="../../plugins/bootstrap-3.3.7-dist/js/bootstrap.min.js"></script>
                                <!-- 图标 -->
                                <link rel="stylesheet" href="../../plugins/font-awesome/css/font-awesome.css" type="text/css"></link>
                                <!-- 公共样式 -->
                                <link rel="stylesheet" href="../../css/eui.css">

实例

                                
                                    <button type="button" class="btn btn_primary">默认</button> 
                                    <button type="button" class="btn btn_primary">点击</button>
