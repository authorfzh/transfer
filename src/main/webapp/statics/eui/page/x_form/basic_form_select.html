基本表单

说明:

  • 引入基本包之外 还需要引入select2的jar包
  • 下拉框用法:根据select定义的id进行初始化，详情见实例
  • select2 的参数常用:Width

select2 api参数

参数 类型 描述
Width 字符串 控制 宽度 样式属性的Select2容器div
placeholder 字符串 选择初始值
multiple 布尔 Select2是否允许选择多个值

此处仅列出常用参数,具体请参考官方文档：http://select2.github.io/select2/

   <!-- 基本引入  -->

初始化下拉框的方法:
$(".select2_demo_1").select2({
width: "300px"
});
$("#smallSelect").select2({
width: "100px"
});
