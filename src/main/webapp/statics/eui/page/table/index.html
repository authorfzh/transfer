分页表格

说明:

  • 表格使用的为layui,文档：https://www.layui.com/doc/modules/table.html#initSort
  • 页面分内部与外部的js,只需修改外部js即可
  • 外部js接口信息:
    (1)初始化加载表格
    (2)向内容方法返回信息
    1:返回cols的json
    2:返回批量操作按钮方法returnToolbarHtml，lay-event="edit" 是操作的标识
    (2)点击事件方法
    1.操作栏的点击事件 toolClick,得到的参数为：event(作为判断的标识),obj(当前行的对象)
    2.尾部按钮事件toolbarClick 该方法为批量才做的事件
    3.行点击事件rowClick

表格初始化

$(function(){
                                                    var params={};
                                                    var paramsJson=
                                                                {
                                                                    "id":"test",
                                                                    "url":"./table.js",
                                                                    "params":params,
                                                                    "method":"post",
                                                                    "page":true,
                                                                    "limit":20,
                                                                    "totalRow":false,
                                                                    "self":false
                                                                }
                                                            
                                                    var table = new LayuiTable(paramsJson);
                                                    table.init();
                                                });
参数键 说明
id 表格id
url 服务请求地址
params 初始化的搜索参数
method 请求类型
page 是否分页 （true/false）
limit 展示分页的条数,默认为10
page 是否分页 （true/false）
totalRow 是否统计
self 是否添加筛选

列的属性

[[  {"type": "checkbox", "align": 'left',"width":"40"}
                                                        ,{"field":"itemId", "title": "常用语标识", "sort": "true" , "align": 'left'}
                                                        ,{"field":"content" , "title": "常用语内容" , "align": 'left','width':'300'}
                                                        ,{"field":"categoryId" , "title": "常用分类名称" ,"templet":categoryIdBtn , "align": 'left'}
                                                        ,{"field":"customerId" , "title": "所属客服" , "align": 'left'}
                                                        ,{"title": "操作","templet":operateBtn , "align": 'left'}
                                                      ]
属性 说明
field 属性名
title 标题名
type checkbox（复选框列），numbers（序号列） 普通列不需设置
fixed 定位 left（固定在左）、right（固定在右）
align 对其方式 left（默认）、center（居中）、right（居右）
templet 扩展方法 (方法的参数为res，需有返回值)

表格解释:对应修改外部js(tableExternal.js),(1)先将数据格式封装成layui的格式(2)定义列属性 是否有特殊转换和操作使用templet方法即可(3)内部透出的点击事件 需注意操作键值判断
