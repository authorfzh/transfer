搜索表格

说明:

  • 表格不多介绍,主要解释高级所搜的用法
  • 该页面数据为静态json，数据库表信息：(1)JS_FILE_CONFIG 字段与用户配置表 (2) JS_SEARCH_CONFIG 用户配置表 (3)JS_SEARCH_FILE 字段详情表
  • 页面分为内部与外部js，外部js首先需要初始化definiteJson 普通搜索条件的json,confJson 配置筛选搜索条件的json,criteriaJson 初始化高级的搜索条件的json,labelJson 标签的json
  • 内部向外部透出的方法:
    (1)saveConf 保存展示列配置
    (2)definiteSerch 普通搜索查询
    (3)advanceedSerch 高级查询
    (4)addLabelJson 处理保存标签
    (5)delLabel 删除标签
    (6)extendClick 字段扩展方法
